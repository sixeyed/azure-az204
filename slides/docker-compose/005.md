# CLI Tools

Two versions, same commands

<div class="mt-8">

<iconify-icon icon="mdi:minus-thick" style="font-size: 2rem; color: #ff6b6b;" />

## docker-compose

**Original tool**
- Separate binary
- Installed alongside Docker
- Uses hyphen

```bash
docker-compose up
docker-compose down
```

</div>

::right::

<div class="mt-8">

<iconify-icon icon="mdi:space-invaders" style="font-size: 2rem; color: #4ecdc4;" />

## docker compose

**Built-in version**
- Part of Docker
- Uses space
- Modern approach

```bash
docker compose up
docker compose down
```

</div>

<div class="mt-12 text-center text-lg">

Commands are identical - just swap hyphen for space

</div>
