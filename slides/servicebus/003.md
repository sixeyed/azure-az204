# What is Service Bus?

<div class="mt-8">

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **High throughput message queuing** - Handle thousands of messages per second

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Reliable storage** - Messages are persisted until they're successfully processed

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Advanced features** - Including dead-letter queues, message sessions, and duplicate detection

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Standard protocol support** - Uses AMQP (Advanced Message Queuing Protocol)

</div>
