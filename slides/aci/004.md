# When to Use ACI

Ideal scenarios

<div class="mt-8">

<iconify-icon icon="mdi:web" class="text-blue-500" /> **Simple web applications**
Single container apps without complex orchestration

<iconify-icon icon="mdi:package-variant" class="text-green-500" /> **Batch processing**
Run tasks on-demand, tear down when complete

<iconify-icon icon="mdi:hammer-wrench" class="text-orange-500" /> **Build agents**
Temporary CI/CD build environments

<iconify-icon icon="mdi:test-tube" class="text-purple-500" /> **Development and testing**
Quick environment spin-up without infrastructure overhead

<iconify-icon icon="mdi:lightning-bolt" class="text-yellow-500" /> **Event-driven applications**
Combined with Azure Functions or Logic Apps

</div>
