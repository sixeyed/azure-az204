# Key Concepts

Important terms to understand

<div class="grid grid-cols-2 gap-6 mt-8">

<div>

### Container Registry

Where images are stored
- Docker Hub (public)
- Azure Container Registry (private)
- Other registries

</div>

<div>

### DNS Name Label

Automatic DNS naming
- Assign friendly prefix
- ACI creates full domain
- Easy access to containers

</div>

<div>

### Resource Allocation

Granular control
- Fractional CPUs possible
- Specific memory amounts
- Multiple cores available

</div>

<div>

### Networking

Port exposure
- Public internet access
- Private networking
- Azure handles complexity

</div>
