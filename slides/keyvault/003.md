# Why Use Key Vault?

<div class="mt-8">

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Encryption at rest** - All data stored in Key Vault is encrypted

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Access control** - You can set fine-grained permissions for who can read which values

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Network isolation** - You can block access to the entire Key Vault except when needed

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Audit logging** - Track every access to your secrets

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Versioning** - Maintain history of secret values for rotation and rollback

</div>
