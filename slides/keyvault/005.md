# URI Structure

```mermaid
graph TB
    URI["https://myvault.vault.azure.net/secrets/db-password/v123"]

    URI --> Base[Vault DNS<br/>myvault.vault.azure.net]
    URI --> Type[Object Type<br/>/secrets/ /keys/ /certificates/]
    URI --> Name[Object Name<br/>db-password]
    URI --> Ver[Version<br/>v123 or latest]
```
