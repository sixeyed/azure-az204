# Why Mock APIs?

<div class="mt-8">

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> Has all the operations agreed upon in the design phase
<iconify-icon icon="mdi:check-circle" class="text-green-500" /> Returns dummy data instead of real data
<iconify-icon icon="mdi:check-circle" class="text-green-500" /> Allows development teams to program against it immediately
<iconify-icon icon="mdi:check-circle" class="text-green-500" /> Prevents delays while waiting for backend implementation

</div>
