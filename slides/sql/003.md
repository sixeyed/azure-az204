# What is Azure SQL?

<div class="mt-8">

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Serverless options** - Where you only pay when the database is in use, perfect for applications with intermittent usage patterns

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **Managed SQL Databases** - Fully managed database as a service with automatic updates and backups

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **SQL Managed Instances** - Near 100% compatibility with on-premises SQL Server

<iconify-icon icon="mdi:check-circle" class="text-green-500" /> **SQL Virtual Machines** - Complete feature parity with SQL Server in your datacenter

</div>
