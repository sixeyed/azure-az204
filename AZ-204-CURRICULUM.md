# AZ-204: Developing Solutions for Microsoft Azure - 2025 Exam Curriculum

## Exam Overview

**Exam Code:** AZ-204
**Certification:** Microsoft Certified: Azure Developer Associate
**Duration:** 100 minutes
**Languages:** Available in 11 languages including English, Japanese, Spanish
**Last Updated:** July 21, 2025
**Renewal:** Annually at no cost through online assessment

## Candidate Profile

The AZ-204 exam is designed for developers who participate in all phases of cloud development including:
- Requirements gathering and design
- Development and deployment
- Security implementation
- Maintenance and performance tuning
- Monitoring and optimization

**Prerequisites:**
- Minimum 2 years of programming experience
- Proficiency with Azure SDKs (Software Development Kits)
- Proficiency with Azure CLI and Azure PowerShell
- Hands-on experience with Azure services

## Skills Measured (Exam Weight Distribution)

### 1. Develop Azure Compute Solutions (25–30%)

#### Implement Containerized Solutions
- Create and manage container images for solutions
- Publish container images to Azure Container Registry
- Run containers using Azure Container Instances
- Create and implement solutions using Azure Container Apps

**Key Technologies:**
- Docker and containerization
- Azure Container Registry (ACR)
- Azure Container Instances (ACI)
- Azure Container Apps

#### Implement Azure App Service Web Apps
- Create and configure Azure App Service
- Enable diagnostics logging
- Configure TLS/SSL settings
- Deploy code and containers to App Service
- Configure autoscaling rules
- Manage deployment slots

**Key Technologies:**
- Azure App Service (Web Apps)
- Application logging and diagnostics
- TLS/SSL certificates
- Deployment slots (staging/production)
- Autoscaling configuration

#### Implement Azure Functions
- Create and configure Function Apps
- Implement input and output bindings
- Implement function triggers
  - Data operations
  - Timer-based triggers
  - Webhook triggers

**Key Technologies:**
- Azure Functions (serverless computing)
- Function bindings and triggers
- Timer triggers
- HTTP/Webhook triggers
- Azure Functions runtime

### 2. Develop for Azure Storage (15–20%)

#### Develop Solutions that use Azure Cosmos DB
- Perform operations on containers and items using the SDK
- Set appropriate consistency levels for operations
- Implement change feed notifications

**Key Technologies:**
- Azure Cosmos DB
- Cosmos DB SDK
- Consistency levels (Strong, Eventual, Session, Bounded Staleness, Consistent Prefix)
- Change feed processor
- Partition keys

#### Develop Solutions that use Azure Blob Storage
- Set and retrieve properties and metadata
- Perform operations on data using the appropriate SDK
- Implement storage policies and lifecycle management
- Implement static website hosting

**Key Technologies:**
- Azure Blob Storage
- Blob SDK operations
- Storage access tiers (Hot, Cool, Archive)
- Lifecycle management policies
- Blob metadata and properties
- Static website hosting

### 3. Implement Azure Security (15–20%)

#### Implement User Authentication and Authorization
- Authenticate and authorize by using the Microsoft Identity platform
- Authenticate and authorize by using Microsoft Entra ID (formerly Azure AD)
- Create and implement shared access signatures
- Implement solutions that interact with Microsoft Graph

**Key Technologies:**
- Microsoft Identity Platform
- Microsoft Entra ID (Azure Active Directory)
- OAuth 2.0 and OpenID Connect
- Shared Access Signatures (SAS)
- Microsoft Graph API
- MSAL (Microsoft Authentication Library)

#### Implement Secure Azure Solutions
- Secure app configuration data using Azure App Configuration and Key Vault
- Develop code that uses keys, secrets, and certificates stored in Azure Key Vault
- Implement Managed Identities for Azure resources

**Key Technologies:**
- Azure Key Vault
- Azure App Configuration
- Managed Identities (System-assigned and User-assigned)
- Key Vault keys, secrets, and certificates
- Azure RBAC (Role-Based Access Control)

### 4. Monitor, Troubleshoot, and Optimize Azure Solutions (5–10%)

#### Implement Caching for Solutions
- Configure cache and expiration policies
- Implement secure and optimized application cache patterns including data sizing, connections, encryption, and expiration

**Key Technologies:**
- Azure Cache for Redis
- Cache policies and patterns
- Cache-aside pattern
- Content Delivery Network (CDN)

#### Troubleshoot Solutions by Using Application Insights
- Configure an app or service to use Application Insights
- Monitor and analyze metrics, logs, and traces
- Implement Application Insights web tests and alerts

**Key Technologies:**
- Azure Application Insights
- Application Performance Management (APM)
- Telemetry data collection
- Live metrics and dashboards
- Availability tests
- Alerts and action groups

### 5. Connect to and Consume Azure Services and Third-Party Services (20–25%)

#### Implement API Management
- Create an Azure API Management instance
- Create and document APIs
- Configure access to APIs
- Implement policies for APIs

**Key Technologies:**
- Azure API Management (APIM)
- API gateways
- API policies (throttling, transformation, caching)
- Developer portal
- OpenAPI specifications

#### Develop Event-Based Solutions
- Implement solutions that use Azure Event Grid
- Implement solutions that use Azure Event Hubs

**Key Technologies:**
- Azure Event Grid
- Event Grid topics and subscriptions
- Event schemas
- Azure Event Hubs
- Event Hubs partitions and consumer groups
- Event-driven architecture

#### Develop Message-Based Solutions
- Implement solutions that use Azure Service Bus
- Implement solutions that use Azure Queue Storage

**Key Technologies:**
- Azure Service Bus (queues and topics)
- Service Bus sessions and transactions
- Dead-letter queues
- Azure Queue Storage
- Message patterns (FIFO, publish-subscribe)

### Programming Languages
- C# or Python or JavaScript/TypeScript (at least one)

### Development Tools
- Visual Studio Code or Visual Studio

- Cost optimization strategies

## Tips for Exam Preparation

1. **Hands-on Practice:** The exam emphasizes practical implementation skills
2. **Use Free Azure Credits:** Azure free tier and credits for new accounts
3. **Focus on SDK Operations:** Know how to use Azure SDKs, not just portal operations
4. **Understand Pricing Models:** Be familiar with how different services are billed
5. **Security First:** Security is integrated across all domains
6. **Practice Troubleshooting:** Use Application Insights and diagnostic tools regularly
7. **Study Official Documentation:** Microsoft Learn is the authoritative source
